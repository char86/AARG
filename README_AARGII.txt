Running AARGII

1) Ensure that AARGI analysis is complete for the experiment to which you wish to apply AARGII

2) Copy the AARGII folder to a convenient location and add it to the Matlab path.

3) Type ‚AARGII‘ into the Matlab command line to bring up the AARGII user interface.

4) Click the help button or watch the AARGII video tutorial for further assistance.

A paper introducing AARG analysis will be published either in a peer-reviewed journal or on bioRxiv.org. Video tutorials and a discussion forum will also be available.

Links will be posted here once these sources are available.
